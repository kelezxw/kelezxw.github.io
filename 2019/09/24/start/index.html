部署到github

1、注册github账号
2、添加ssh key
3、在github上新建项目
注意你的项目名为 username.github.io 其中username是你github的账户名
4、配置本地项目
配置文件是项目根目录下 _config.yml

5、推送文章上去

现在你可以用kelezxw.github.io访问你的博客了

配置私有域名

1、首先你要买个域名
2、域名指向你github域名的ip
3、source文件夹下新建 CNAME 文件，没有后缀，里面写上你的域名
4、发布项目，过一会你就可以通过自己的域名访问了
